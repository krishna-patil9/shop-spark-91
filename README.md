# ShopSpark-91

A modern full-stack e-commerce web application built to provide a seamless online shopping experience. The platform includes secure user authentication, product browsing, cart management, order processing, and an intuitive admin workflow.

## Features

* User authentication and authorization
* Product browsing with search and filtering
* Shopping cart and checkout flow
* Secure payment integration
* Order management
* Responsive UI for desktop and mobile
* Modern and scalable component-based architecture

## Tech Stack

**Frontend**

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui

**Backend**

* FastAPI
* REST APIs

**Database**

* Supabase (PostgreSQL)

## Getting Started

### Prerequisites

* Node.js (v18 or later)
* npm

### Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd ShopSpark-91
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
